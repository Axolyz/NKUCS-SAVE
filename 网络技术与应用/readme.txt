# 这课我没选，以下也是前人评价
这门课是网络课的延伸，课上讲的比较细，但作业相对多一点。
课上会随机做题，好像是六次（记不太清了）。提交后当作检测成绩。
实验课不会讲知识，如果见面课讲不完会占用一定的实验课时间，一般不会太长。（这里吐槽一下去年的课表，理论课放在6 7节，实验课在9 10，搞得中午没法睡觉，没法选中午的课，第八节课也很鸡肋，希望今年好一点）
实验课的实验会比较简单，按照教材的步骤来就行，做完要交实验报告。
基本每周都有实验，除了正常的柯本实验，还会有编程实验，不比网络的作业简单。去年的助教非常和蔼可亲，最后基本都是满分。
除了理论课和实验课，张老师还有自己的网课，要在雨课堂看，时长与成绩挂钩。最后一节课有一个考试和网课相关，好像是十五分钟，三十道选择（应该都是选择题，时长和数量记不太清了）
考试周没有安排考试。
张老师课上喜欢和同学们交流，如果回答问题会有记录，加分，建议会不会都回答一下。

最后下来成绩组成有：课堂小测、实验报告、大作业们、网课时长、最后一节课的测试、课堂回答问题的次数。

啊！差点儿忘了，老师有点名，没来直接从总成绩扣分，5-6分左右，没有解释机会，任何请假条都没用。——去年带军训，第一节没去上，被点到的人如是说。
